# Statistical Analysis: Drug A (treatment) Effect on Reaction Time

## Hypothesis (H0):
Drug A has no effect on reaction time compared to saline control group?

## Project Overview
1. Create a sample dataset with:
   - Control group
   - Treatment group
2. Import the dataset into RStudio
3. Check the datastructure
4. Performed an independent two_sample t_test

## Results
A significant difference was found (t(7.82) = 5.48, p = 0.00065), providing initial evidence for the compound's effect. Full reproduced code in both R and Python is provided.

## Statistical Method
The analysis compares reaction time means between a Control and Treatment group using an independent Welch's t-test
